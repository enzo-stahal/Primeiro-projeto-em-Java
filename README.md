#  Meu Primeiro Projeto em Java

Este repositório reúne meus primeiros exercícios e projetos desenvolvidos em **Java**, com foco no aprendizado dos conceitos fundamentais da linguagem e da lógica de programação.

## 📚 Conteúdo do Projeto

### 🔹 Exercícios Básicos de Java
Códigos voltados para a prática dos conceitos iniciais da programação, incluindo:

- Entrada e saída de dados
- Variáveis e tipos primitivos
- Operadores matemáticos
- Estruturas condicionais (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`, `do-while`)
- Manipulação básica de dados

### 🔹 Expressões Booleanas
Exercícios para praticar lógica booleana e tomada de decisões:

- Operadores relacionais (`>`, `<`, `>=`, `<=`, `==`, `!=`)
- Operadores lógicos (`&&`, `||`, `!`)
- Avaliação de condições
- Fluxo de execução baseado em decisões

### 🎮 Jogo de Adivinhação
Um jogo simples em que o usuário tenta adivinhar um número aleatório entre **0 e 100**.

#### Funcionalidades:
- Geração aleatória de números
- Dicas indicando se o número é maior ou menor
- Contagem de tentativas
- Mensagem de vitória ao acertar

### 🏦 Sistema Bancário Simples
Projeto desenvolvido para praticar variáveis, condicionais e interação com o usuário.

#### Funcionalidades:
- Exibição dos dados do cliente
- Exibição do tipo da conta
- Consulta de saldo
- Recebimento de transferências
- Realização de transferências
- Atualização automática do saldo

## 🎯 Objetivo

Este projeto foi criado com o objetivo de consolidar os conhecimentos iniciais em:

- Lógica de programação
- Programação orientada a objetos (conceitos iniciais)
- Estruturas de controle
- Manipulação de dados em Java
- Desenvolvimento de pequenos sistemas de console

## 🛠️ Tecnologias Utilizadas

- Java
- IDE de sua preferência (IntelliJ IDEA, Eclipse ou VS Code)
